# Sistema de Inventario y Gestión — Creaciones Repiso

Este proyecto es una aplicación en Python para gestionar inventario, ventas, arreglos y abonos en un taller de calzado.

## Características
- Gestión de inventario
- Registro de ventas
- Registro de ingresos de productos
- Registro de arreglos y remodelaciones
- Control de abonos
- Estados: pendiente, pagado, retirado
- Registro mensual de actividades

## Cómo ejecutar
1. Tener Python 3 instalado.
2. Ejecutar el archivo:
   python ProyectoPC01.py

## Menú principal
1. Mostrar inventario  
2. Registrar venta  
3. Registrar ingreso de producto  
4. Registrar arreglo o remodelación  
5. Ver lista de arreglos  
6. Marcar calzado como retirado  
7. Registrar abono de cliente  
8. Ver registro mensual  
9. Salir
